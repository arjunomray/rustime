#+TITLE: README


* Rustime

It is just a simple to use cli based pomodoro timer.

** Usage commands:
#+BEGIN_SRC  shell
->rustime -h
Usage: rustime [OPTIONS]

Options:
  -s, --session-time <SESSION_TIME>    [default: 25]
  -b, --break-time <BREAK_TIME>        [default: 5]
  -n, --session-title <SESSION_TITLE>  [default: "Working on Stuff"]
  -h, --help                           Print help
#+END_SRC
